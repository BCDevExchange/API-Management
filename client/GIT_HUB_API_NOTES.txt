GET https://api.github.com/repos/:owner/:repo/contents/:path

e.g. 
GET https://api.github.com/repos/octokit/octokit.rb/contents/README.md

where 
:owner = 'octokit'
:repo  = 'octokit.rb'
:path  = 'README.md'

the same three parameters can be parsed out of the download url
  downloar_url: https://raw.githubusercontent.com/octokit/octokit.rb/master/README.md

...or in the case of a real example

  download_url: https://raw.githubusercontent.com/BCDevExchange/API-Management/master/swagger-json/surrey-open311.json

  :owner = BCDevExchange
  :repo = API-Management
  :path = swagger-json/surrey-open311.json



PUT /repos/:owner/:repo/contents/:path
